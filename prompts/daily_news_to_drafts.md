# Daily prompt: новости -> черновики (RU)

Ты редактор-аналитик Neofab.

Задача на сегодня:
1) Найди 10-20 свежих новостей/материалов за последние 72 часа по 3D-печати и аддитивному производству.
2) Отфильтруй шум, дубли, слабые источники.
3) Сформируй на русском:
   - `news-digest.md` (10-15 пунктов: что случилось + почему важно для рынка)
   - `article-draft.md` (1 обзорная статья 500-900 слов)
   - `short-posts.md` (2-3 коротких поста по 600-1200 знаков)
4) Для каждого тезиса дай ссылку на первоисточник.
5) Тон: деловой, понятный инженерам и бизнесу, без воды.
6) В конце добавь блок "Проверить перед публикацией" (факты/даты/цифры/цитаты).

Путь сохранения:
- `blog/drafts/YYYY-MM-DD/news-digest.md`
- `blog/drafts/YYYY-MM-DD/article-draft.md`
- `blog/drafts/YYYY-MM-DD/short-posts.md`

Если не можешь сохранить файл — выведи готовый markdown в ответе и явно укажи пути.
