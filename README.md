# Neofab Blog Drafts

Черновики и контент-план для позиционирования Neofab как информационного центра по 3D-печати.

## Структура

- `sources.md` — авторитетные источники для мониторинга
- `prompts/` — рабочие промпты для агентов
- `drafts/YYYY-MM-DD/` — ежедневные черновики
- `calendar/` — контент-календарь

## Формат ежедневного цикла

1. Сбор ключевых новостей за сутки
2. Фильтрация дублей и нерелевантного
3. Короткие выжимки на русском
4. Черновик обзорной статьи + 2-3 коротких поста
5. Список первоисточников

## Важно

Публикация только после ручной валидации фактов.
